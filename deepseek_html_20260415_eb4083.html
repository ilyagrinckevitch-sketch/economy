Детали расчёта IRR

Внутренняя норма доходности (IRR) — ставка, при которой NPV = 0. Для проекта IRR составляет ~33%, что значительно выше ставки дисконтирования (24%). Проект устойчив к повышению стоимости капитала.

Ставка 24% → NPV = +1,71 млн ₽Ставка 33% → NPV ≈ 0Ставка 35% → NPV = -0,17 млн ₽

Все расчёты основаны на данных из приложенной финансовой модели (инвестиционные затраты, операционные расходы, консервативный прогноз привлечения специалистов и предпринимателей).
